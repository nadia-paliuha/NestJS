# Restaurant Table Booking System

Система для **бронювання столів у ресторані**, розроблена на **NestJS (TypeScript)** з використанням **PostgreSQL** та **JWT-аутентифікації**.  
Мета — створити зручний REST API для користувачів і адміністраторів ресторану.

## Опис проєкту

Користувачі можуть:
- Реєструватися та входити в систему;
- Переглядати доступні столи;
- Бронювати стіл на дату й час, вказуючи кількість людей;
- Переглядати та скасовувати власні бронювання.

Адміністратори можуть:
- Керувати списком столів (CRUD);
- Переглядати всі бронювання;
- Підтверджувати або скасовувати бронювання користувачів.

---

## Функціональні можливості

- **Реєстрація та вхід користувачів** 
  - через **Google OAuth 2.0** (авторизація за допомогою Google акаунту)

- **Особистий кабінет користувача**
  - перегляд власних бронювань  
  - скасування бронювання  
  - редагування профілю

- **Бронювання столів**
  - вибір дати, часу та кількості гостей  
  - перевірка доступних столів  
  - автоматична перевірка перетинів за часом  
  - підтвердження або скасування броні адміністратором  

- **Керування столами (для адміністратора)**
  - створення, редагування та видалення столів  
  - вказання місткості та розташування  
  - активація/деактивація столів  

- **Керування бронюваннями (для адміністратора)**
  - перегляд усіх бронювань користувачів  
  - зміна статусу (підтверджено / відхилено / скасовано)  
  - фільтрація бронювань за датою, користувачем або статусом
